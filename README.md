## Extension Ideas

1. Post Versioning (Very Impressive Feature)

Allow posts/docs to have version history. Example:

```
/posts/{id}/versions
```

Features:
- Track edit history
- View previous versions
- Restore old versions
- Implementation ideas:
- Separate post_versions table
- Store diff or full snapshots

This shows data modeling and real-world thinking.
